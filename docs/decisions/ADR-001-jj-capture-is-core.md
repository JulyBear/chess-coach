# ADR-001：项目唯一核心能力是 JJ 自动采谱

## 状态
已采纳

## 背景
当前项目中，真正不可替代、必须自研的部分，不是棋盘展示、不是引擎分析、也不是 AI 复盘，而是 **JJ 象棋对局记录的自动获取**。

其他能力都已有成熟组件可以复用。

## 决策
将 `proxy/jj_addon.py` 所代表的 JJ 自动采谱链路定义为项目唯一核心自研能力。

## 原因
- JJ 协议抓取是项目差异化来源。
- 其余模块均可通过成熟开源组件或服务完成。
- 这样可以把工程精力集中在最有价值的地方。

## 放弃的方案
- 把前端棋盘、规则、分析、AI 解说都当作自研核心一起做。

## 影响
- 后续优先保证采谱链路稳定。
- 其它模块优先复用，不轻易自写复杂实现。
