# ADR-004：创新主线聚焦训练题库、布局学习与个性化强化

## 状态
已采纳

## 背景
用户明确提出，产品的真正创新不在基础复盘能力，而在 AI 引入之后的新训练产品形态：
- 常见杀法训练
- 题库自然语言调用
- 定式与布局学习
- 用户个人错误总结
- 个性化强化训练

基础复盘能力只是底座，不是最终壁垒。

## 决策
产品创新主线聚焦在三个方向：
1. 杀法/残局/战术题库训练
2. 定式/布局学习与熟练掌握
3. 基于个人棋谱错误模式的个性化强化学习

## 原因
- 这些能力能形成长期学习闭环。
- 能把用户自己的棋谱转成持续训练价值。
- 比单次复盘报告更容易形成留存与差异化。

## 放弃的方案
- 只做一个普通复盘工具
- 只堆叠引擎分数和 AI 总结，不形成训练闭环

## 影响
- 后续需要新增题库、布局、掌握度、弱点画像等数据模型。
- 前端后续要从单一复盘页扩展为复盘 + 训练 + 学习路径。
- LLM 不仅用于复盘报告，也用于学习引导、自然语言找题、弱点强化建议。
